# 0528_Processing

5 月 28 日のメディア表現基礎の授業で Processing を取り扱ったので、授業内容を少し応用してみました。

## 概要

p5.js と Matter.js を使ってフィールド内をクリックしている間にボールが生成され、そのボールがフィールド内にある数とその最高記録をカウントするゲームを作りました。

## 遊び方(ビルド方法)

1. ビルドに必要なパッケージをインストールします

   ```shell
    npm i
   ```

2. ビルドします
   ```shell
   npm run prod
   ```
3. `./dist/index.html`を開くと遊べます
